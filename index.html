mdbookプロジェクトを作成

cd Desktop

mkdir mdbook_test

cd mdbook_test

リポジトリに登録するためのSSH鍵(mdbook_testにファイルで保存される)あとで消す

ssh-keygen -t rsa -b 4096 -f gh-pages
or
ssh-keygen -t rsa -b 4096 -C "$(git config user.email)" -f gh-pages -N ""

githubレポジトリを２つ作成,1つはmdbookを作成する プロジェクト 用,1つは GitHub Pages用

[mdbookを作成する プロジェクト 用の作業]

Settings > Deploy keysに任意の名前で登録します(今回の名前は"ACTIONS_DEPLOY_KEY")

keyに(gh-pages.pub)をペースト

Allow write access にチェックを入れる(このキーでこのリポジトリにプッシュできますか？デプロイキーは常にプルアクセスが可能です。)

[GitHub Pages用]

Settings > Secretsに"ACTIONS_DEPLOY_KEY"という名前で登録します

keyに(gh-pages)をペースト

mdbook_testに保存されている(gh-pages.pub)と(gh-pages)は削除して良い
